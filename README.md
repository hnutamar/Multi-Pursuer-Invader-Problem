# Multiple Pursuer Invader problem
This project explores problem of defending moving target against possible invaders using vortex field approach.
## Simulator
To create virtual environment with dependencies, run:
```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
To use the simulator, run:
```
cd src/
python sim2D.py #2D simulator
python sim3D.py #3D simulator
```
